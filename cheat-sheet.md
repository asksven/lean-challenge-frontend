### Create a new project 
`ng new <project-name> [--directory <dir-name>] [--prefix <project-prefix>]`
Prefix is recommended to avoid any namespace-clashes
### Create a component
`ng g c <component-name> [--flat]`
### Create a directive
`ng g d <directive-name> [--flat]`
### Create a class
`ng g cl <class-name>`
### Create a service
`ng g s <service-name>`
### Create a pipe
`ng g pipe`
